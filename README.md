# 📦 Controle de Estoque

Um sistema simples e eficiente desenvolvido em Python para registrar a entrada e a saída de produtos, permitindo acompanhar a movimentação e atualizar o estoque disponível em tempo real.

Este projeto foi desenvolvido como atividade prática para consolidação de conceitos de lógica de programação com Python, versionamento de código com Git e publicação de repositórios no GitHub.

---

## 🚀 Funcionalidades

- **Cadastro Inicial:** Solicita o nome do produto e a quantidade inicial em estoque.
- **Movimentação Dinâmica:** Permite registrar operações de **entrada** (abastecimento) ou **saída** (vendas/retiradas).
- **Validação de Regras de Negócio:**
  - Impede que o estoque inicial seja negativo.
  - Impede movimentações com valores menores ou iguais a zero.
  - Bloqueia saídas que excedam o estoque disponível (evita estoque negativo).
  - Valida se o tipo de movimentação digitado é válido.
- **Visualização:** Apresenta o saldo de estoque atualizado após cada operação bem-sucedida.
